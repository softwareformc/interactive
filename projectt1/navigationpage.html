Level One Restrictions for level one: Make 5 variations of your list using only HTML. In addition to the required html, head, title, and body elements, you can use any of the following HTML elements: paragraph, line break, and space.
Level Two Restrictions for level two: Only use HTML. In addition to the elements allowed in Level 1, you can use any of the following HTML elements:
- div (block element)
- span (inline element)
- h1, h2, h3, h4, h5, h6 (headings)
- strong (strong text)
- em (emphasized text)
- ol, ul, li (lists)
- a (links: do not link to external sites, but you can link to parts of your poem using an attribute to assign an id)
Again, you can use & n b s p ; to equal a visible space.
Level Three Restrictions for level three: Use both HTML and CSS. Only use the HTML allowed in Levels 1 and 2. Only use the following CSS properties:
- padding
- margin
- border
- position
- float
- top, right, bottom, left
- height, width
Level Four Restrictions for level four: Use both HTML and CSS.Only use the HTML allowed in Levels 1 and 2. In addition to the CSS properties allowed in Level 3, you can use any of the following CSS properties:
- font-style
- font-weight
- font-size
- font-family
- text-align
- line-height
- letter-spacing
- text-transform
Level Five Restrictions for level five: Use both HTML and CSS. Use any HTML element, except for img, iframe, and a via linking to external sites. Use any CSS property

This project is an iterative exploration of typography through HTML and CSS. My list of 26 words was selected from Steering the Craft, a creative writing guide by Ursula K. Le Guin published in 1998. There are 25 different representations of this list, each on a different web page. The 25 variations are divided into 5 levels of coding restraints, which gradually allow more freedom.
